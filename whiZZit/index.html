StackOverflow is any developer's best friend and if you encounter a problem somewhere there's always a person that's solved it before you. But there is no incentive for the expert to answer the problem on Stack, so we set out to change that. How do we change that? StackOverflow with a bounty! Earn some money for your skill and pay back those tuition fees

But we didn't stop there, we integrated it with the blockchain so well that we do not need to store the questions pon our servers, instead we just stored them on the blockchain. We also integrated it with the rocket chat api that allowed us to create a chat instance between the user and the expert.

What's next? Move it from the rinkeby test network to the main ethereum blockchain
